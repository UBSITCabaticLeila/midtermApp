Two-Way Binding

Two-Way Binding [( )]

Synchronizes data between the component and the template in both directions using [(ngModel)].

Syntax: [(ngModel)]="property"

When the input changes, the component property updates. When the component property changes, the input updates. It combines [property] and (event) binding in one.

D — Two-Way Binding [( )]

Two-way binding allows you to synchronize data between the component and the template in both directions.

When you use two-way binding with [(ngModel)], any changes made to the input field will automatically update the corresponding property in the component, and any changes to that property in the component will automatically update the input field in the template.
