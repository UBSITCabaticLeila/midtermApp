One-Way Binding

Event Binding ( )

Listen and respond to user events such as clicks, input changes, and form submissions. Data flows from the view to the component.

Syntax: (event)="handler()"

event is the DOM event to listen for (e.g. click, input, submit), and handler is the component method called when it fires.

Event binding triggers component logic in response to user interactions — updating state, processing input, or performing actions.
