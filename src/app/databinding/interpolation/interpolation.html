One-Way Binding

String Interpolation {{ '{{' }} {{ '}}' }}

Display dynamic data from your component's TypeScript code within your HTML template using double curly braces.

Syntax: {{ '{{' }} expression {{ '}}' }}

Angular evaluates the expression inside the curly braces and inserts the result as a text node in the DOM. Data flows from the component to the template.

Rules: Must return a value. You CAN call pure methods and use pipes. You CANNOT use assignments (=), increment (++), or access global objects like window.

A
Basic Interpolation

Direct property access, arithmetic, method calls, and ternary expressions inside {{ '{{' }} {{ '}}' }}.

{{ '{{' }} title {{ '}}' }}

Live Output

Title{{ title }}
Student{{ studentName }}
Course{{ courseCode }}
Next Year{{ currentYear + 1 }}
Welcome{{ getWelcomeMessage() }}
Status{{ isActive ? 'ACTIVE' : 'INACTIVE' }}
B
Interpolating Objects

Access object properties using dot notation and safe navigation operator ?. to prevent null errors.

{{ '{{' }} user.name {{ '}}' }}

Live Output

Name{{ user.name }}
Position{{ user.position }}
Department{{ user.department }}
Role{{ user.details?.role }}
C
Iterating with Interpolation

Use @for to loop through an array and display each item using interpolation.

@for (skill of skills; track skill)

Live Output

@for (skill of skills; track skill) { {{ skill }} }
D
Safe Navigation & Select

The ?. operator safely returns empty string if value is null or undefined instead of throwing an error.

{{ '{{' }} user.details?.role {{ '}}' }}

Live Output

User Name{{ user.name }}
User Role{{ user.details?.role }}
D2
Arrays of Objects

Looping through an array of objects and binding selected value using [(ngModel)].

@for (c of countries; track c.id)

Live Output

Selected ID: {{ selectedCountryId }}

E
Debugging with JSON Pipe

Display the entire object structure as a JSON string using the json pipe — useful for debugging.

{{ '{{' }} user | json {{ '}}' }}

Live Output

{{ user | json }}
